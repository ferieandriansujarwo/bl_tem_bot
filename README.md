
# BL TE Bot
Service for TEM telegram bot bukalapak (@bl_tem_bot)

## Prerequisites
1. Node.js >= 6.2.0

## Install local machine

1. Clone repo
```
git clone git@github.com:ferieandriansujarwo/bl_tem_bot.git
```
2. make `.env`
```
cp env.sample .env
```
3. Install dependencies
```
npm install
```
4. Run Bot
```
npm run start
```
